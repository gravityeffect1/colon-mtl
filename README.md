# colon-mtl

Proiect de computational pathology — clasificare și segmentare 
a leziunilor colorectale din imagini WSI (Whole Slide Images).

## Dataset
- 31 slide-uri colon de la Colentina (4 clase: ADK-14, HG-10, LG-5, Hplastic-3)
- Date suplimentare: TCGA-COAD (în lucru)

## Structura
- `src/preprocessing/` — audit și extracție tile-uri
- `src/data/` — Dataset classes
- `src/models/` — arhitecturi
- `configs/` — configurații YAML
